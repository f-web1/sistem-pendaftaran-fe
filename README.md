# Frontend
# 🚀 Sistem Pendaftaran Mahasiswa Baru Berbasis Web (Frontend)

Repositori ini merupakan bagian **Frontend** dari tugas pengembangan aplikasi web sistem pendaftaran mahasiswa baru.

## 👥 Informasi Kelompok 3
* **UI/UX:** Fanzhi Sukma
* **Frontend:** Fahri Ahmad Subhakti
* **Backend:** Tajul Fazarie

## 🛠️ Tech Stack
* **Runtime/Framework:** React.js (Vite)
* **Styling:** Tailwind CSS

## ⚙️ Cara Menjalankan Proyek
1. Clone repositori ini.
2. Jalankan perintah `npm install` untuk memasang semua dependencies.
3. Jalankan `npm run dev` untuk memulai local development server.
